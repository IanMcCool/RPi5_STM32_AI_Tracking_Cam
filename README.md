# RPi5_STM32_AI_Tracking_Cam
Real-time object detection using Hailo-8L NPU on Raspberry Pi 5
GStreamer + Hailo SDK Python pipeline with OpenCV frame overlay 
USB serial communication bridging RPI5 host and STM32F4 Proportional servo control with configurable gain and dead-band filtering
2-DOF pan-tilt gimbal driven with MG996R servos. 
Mbed OS firmware (C++) STM32 Nucleo F4 series
