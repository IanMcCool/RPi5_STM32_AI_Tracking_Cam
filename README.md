# RPi5_STM32_AI_Tracking_Cam
Real-time pan-tilt object tracking system combining edge AI inference on a Raspberry Pi 5 with embedded servo control on an STM32F4 microcontroller.
Rasberry Pi 5 is connected to OV5647 camera module and also has the HAILO 8 AI HAT+ 26 TOPS to offload inference workload from CPU, boosting FPS, and inference rate
STM32F4 nucleo board setup is connected to two mg90s servos for pan and tilt tracking
